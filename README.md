<img src="https://img.shields.io/badge/Version-1.5.0-e94560?style=for-the-badge" />

> **TalkHub** is an enterprise-grade video conferencing and team collaboration platform with real-time audio/video streaming, team chat, screen sharing, whiteboard collaboration, and scheduled meetings — built from scratch across multiple platforms.

### 🏗️ Architecture

```
┌──────────────────────────────────────────────────────────────────┐
│                        TalkHub Platform                          │
├──────────────┬──────────────┬──────────────┬─────────────────────┤
│  🖥️ Desktop  │  📱 Mobile   │  🌐 Web UI   │  🔧 Admin Panel    │
│  WPF (.NET)  │  Flutter     │  Flutter Web │  ASP.NET MVC       │
└──────┬───────┴──────┬───────┴──────┬───────┴──────────┬──────────┘
       │              │              │                  │
       └──────────────┴──────────────┴──────────────────┘
                              │
                    ┌─────────┴─────────┐
                    │                   │
              ┌─────▼──────┐     ┌──────▼────────┐
              │  REST API   │     │  TCP Server   │
              │  ASP.NET    │     │  .NET 8.0     │
              │  Web API 2  │     │  Binary Proto │
              └─────┬───────┘     └───────────────┘
                    │
              ┌─────▼──────┐
              │ SQL Server  │
              │  Database   │
              └─────────────┘
```


### ✨ Key Features

<table>
<tr>
<td width="50%">

**🎥 Video Conferencing**
- HD video & audio calls
- Adaptive grid layout (1-16+ participants)
- Screen sharing with live preview
- Dynamic Island recording indicator
- Meeting recording with save indicator

</td>
<td width="50%">

**💬 Team Communication**
- Real-time room chat with attachments
- Team channels (public & private)
- Direct messaging with read receipts
- Emoji reactions & hand raise
- File sharing (images, docs, media)

</td>
</tr>
<tr>
<td width="50%">

**🎨 Whiteboard Collaboration**
- 11 drawing tools (pen, shapes, text, etc.)
- Lasso selection & drag elements
- Sticky notes & comments
- Laser pointer with fade effect
- Export & share to chatroom

</td>
<td width="50%">

**📅 Meetings & Management**
- Scheduled meetings with calendar
- Meeting invitations & reminders
- Host/co-host controls
- Participant permissions management
- Kick, mute, and admit controls

</td>
</tr>
<tr>
<td width="50%">

**🔐 Security & Auth**

- Session token authentication
- Role-based access control
- Account expiration system
- 12-month trial management

</td>
<td width="50%">

**📦 Deployment & Distribution**
- Professional Inno Setup installer
- Custom recordings path selection
- URL protocol (talkhub:// links)
- Auto-update mechanism
- Admin panel for management

### 🧰 Tech Stack Used

<div align="center">

| Layer | Technology | Purpose |
|:---:|:---:|:---:|
| 🖥️ Desktop | WPF + .NET 8.0 | Windows desktop client |
| 📱 Mobile | Flutter + Dart | Cross-platform mobile app |
| 🌐 Admin | ASP.NET MVC 5 + Metronic | Admin web panel |
| ⚡ API | ASP.NET Web API 2 | REST API server |
| 🔌 Realtime | TCP Sockets + Binary Protocol | Audio/video streaming |
| 🗄️ Database | SQL Server + EF6 | Data persistence |
| 📦 Installer | Inno Setup | Windows installer |

</div>


### 📂 Project Structure

```
TalkHub/
├── 🔌 talkHUBAPI2/          → REST API Server (ASP.NET Web API 2)
├── 📡 talkhub-server/        → TCP Signaling Server (.NET 8.0)
├── 📚 TalkHub.BL/            → Business Logic Layer (EF6 Code-First)
├── 🖥️ talkhub-wfp/           → Windows Desktop Client (WPF)
│   └── Setup/                → Inno Setup Installer
├── 📱 talkhub-flutter/       → Mobile & Web Client (Flutter)
├── 🔧 Talkhub.WebUI4.8/     → Admin Panel (ASP.NET MVC)
├── 🗄️ Database/              → SQL Server Scripts
└── 📦 TalkHub.Common/        → Shared Library
```
